# Your Memory 🧠

**Your Memory** 是一款基于 Vue 3 和 Golang 开发的跨平台个人记忆管理系统。它不仅是一个笔记工具，更是你的“第二大脑”。通过集成 AI RAG（检索增强生成）技术，它能读懂你的笔记，并随时回答你的提问。

## ✨ 核心特性
* 🚀 **三端互通**：支持 Web、移动端适配及桌面端同步。
* 🤖 **AI 记忆检索**：基于 RAG 技术，AI 结合你的个人笔记提供精准回答。
* ⚡ **极致性能**：后端采用 Go 语言编写，响应迅速，部署轻量。
* 🔐 **隐私安全**：支持本地向量存储，你的记忆由你掌控。

## 🛠 技术栈
- **Frontend:** Vue 3, Vite, Tailwind CSS, Pinia
- **Backend:** Golang, Gin, GORM
- **Database:** PostgreSQL (with pgvector)
- **AI Integration:** OpenAI / DeepSeek / Ollama (LangChainGo)

## 🏗 快速开始

1. 时代背景：信息过载与记忆碎片化
在信息化社会，人们每天接收的信息量呈爆炸式增长。无论是学习中的专业知识，还是工作中的项目细节，都以碎片化的形式散落在各种设备中。传统的“大脑记忆”已难以应对如此海量的信息，人类对“第二大脑”（数字记忆库）的需求日益迫切。

2. 现状分析：传统笔记系统的局限性
虽然市场上已有 Notion、Obsidian、微信收藏等成熟产品，但它们仍存在以下痛点：

检索困难： 当笔记积累到成千上万条时，基于关键词的传统搜索往往难以精准定位信息，用户经常“记了就忘，找了半天”。

数据孤岛： 不同端的数据同步体验不一，且难以将静态的笔记内容转化为动态的知识解答。

缺乏交互： 笔记仅仅是死板的文本，无法主动与用户对话，用户需要手动翻阅、总结和归纳，效率低下。

3. 技术契机：AI 与 RAG 技术的兴起
近年来，大语言模型（LLM）的突破为知识管理带来了新思路。通过 RAG（检索增强生成） 技术，我们可以将个人私有的笔记数据向量化，让 AI 能够检索并学习这些特定知识。这意味着，笔记库不再只是一个“仓库”，而是一个“智能助理”，能够基于用户的真实经历和知识储备回答问题。

4. 研究意义与目的
本项目旨在开发一款名为 Your Memory 的跨平台记忆管理系统，采用 Vue 3 和 Golang 构建高性能的前后端架构。其核心目标是：

实现无缝同步： 解决学习与工作场景下多端协作的需求。

引入智能交互： 利用 AI 技术激活沉睡的笔记，实现从“存储信息”到“理解知识”的跨越。

开源与私有化： 为个人用户提供一个安全、可扩展的开源解决方案，探索 AI 在个人知识管理领域的应用实践。
